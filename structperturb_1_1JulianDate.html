Detailed Description

Represents a specific point in time on the Julian calendar.

Generally not constructed manually, but instead converted from a YMDhms or from Satellite::epoch. Supports some basic manipulation operations. For a human readable representation, can be converted back to YMDhms. As for what time point this represents, see the comment on YMDhms.
